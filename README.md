## Инструкция по сборке и запуску проекта:

1. Выполняем: docker-compose up -d
2. После сборки и запуска контейнера, выполняем sudo chmod 777 -R ./
3. Копируем файл .env.example, вставляем в корень проекта как .env
4. Заходим в контейнер приложения docker exec -it app_tt bash
5. Далее выполняем по очереди:   
6. php artisan generate:key 
7. php artisan migrate

## Приложение должно быть доступно по адресу: 

### http://127.0.0.1:8008
