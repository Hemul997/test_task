## Инструкция по сборке и запуску проекта:

1. Выполняем: `docker-compose up -d`
2. После сборки и запуска контейнера, выполняем в корне проекта `sudo chmod 777 -R .`
3. Заходим в контейнер приложения: `docker exec -it app_tt bash`
4. Выполняем `composer setup`

## Приложение должно быть доступно по адресу: 

### http://127.0.0.1:8008
